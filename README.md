# MyLoci GitHub Pages

This folder contains the public website for App Store Connect:

- `/privacy/` for the required Privacy Policy URL
- `/support/` for the App Store Support URL

Before publishing, replace `support@example.com` in `support/index.html` with the real support email address.

GitHub Pages setup:

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings -> Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select the main branch and the `/docs` folder.
6. Save.

The URLs will usually be:

- `https://davidmetela.github.io/YOUR_REPOSITORY_NAME/privacy/`
- `https://davidmetela.github.io/YOUR_REPOSITORY_NAME/support/`
